# Agentic AI Projects Portfolio

# What Makes This Different
This portfolio focuses on building **agentic AI systems**, not just LLM wrappers.

The projects demonstrate:
- LLM-based decision making  
- Tool execution (code + APIs)  
- Retrieval-augmented generation (RAG)  
- End-to-end deployment (FastAPI + cloud)  

These systems simulate how modern AI assistants and enterprise AI tools are built.


# System Design Understanding

Across these projects, I implemented:

- LLM-driven decision layers  
- Tool abstraction (code execution, retrieval, APIs)  
- Workflow orchestration  
- Separation of reasoning vs execution  

These reflect real-world agent system design patterns.
